# editor-plugin-life-safety-plan

Editor plugin to generate life safety plans

## Development
### Local Development

```bash
# run the app
npm run dev

# create a build
npm run build
```

### Tests

tests are run for pull requests

```bash
# unit tests
npm run test

# typescript test
npm run typecheck
```

### Release

To create a new release and updated changelog run:

```bash
npm run release
```
